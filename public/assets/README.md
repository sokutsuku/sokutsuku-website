# Assets Directory

このディレクトリには、その他のアセットファイルを保管します。

## 対象ファイル

### フォントファイル
- カスタムフォント（.woff2, .woff, .ttf）
- アイコンフォント

### 設定ファイル
- manifest.json
- robots.txt
- sitemap.xml

### その他
- ファビコン関連ファイル
- PWA用アイコン
- OGP画像

## ディレクトリ構成例

```
assets/
├── fonts/
│   ├── custom-font.woff2
│   └── icon-font.woff
├── favicons/
│   ├── favicon.ico
│   ├── apple-touch-icon.png
│   └── android-chrome-512x512.png
└── manifest.json
```
